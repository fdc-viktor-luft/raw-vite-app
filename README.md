# raw-vite-app
Initial setup on top of `vite` and template `react-ts` with some dummy functionality. See
[here](https://vitejs.dev/guide/) to learn more about `vite`.

Including:
- `jest`
- `eslint`
- `prettier`
- `madge`
- `sass`
- script targets `pnpm lint` and `pnpm test`
- GitHub Workflow
